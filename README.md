# Water Clean-Up Adventure 🌊

An independently built environmental awareness game app developed with vanilla JavaScript, HTML, and CSS.

## About
A browser-based game designed to raise awareness about ocean pollution. Players collect trash from the ocean while avoiding obstacles, with self-implemented game logic and state management.

## Features
- Self-implemented game loop and state management
- Score tracking and level progression
- Responsive design for desktop and mobile

## Tech Stack
- JavaScript (vanilla)
- HTML / CSS
- Git / GitHub
